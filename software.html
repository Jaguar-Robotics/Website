What We Do

Our Role.


The software subteam programs everything the robot does — from precise swerve drive control and autonomous navigation to computer vision and real-time telemetry. We write Java using WPILib and deploy on a RoboRIO, competing against hundreds of teams with code we built from scratch every season.

Our code is open source on GitHub at github.com/Jaguar-Robotics. We believe in sharing knowledge with the FRC community — every season's code is publicly released after competition.

Responsibilities

What We Build.

🤖
Robot Control
Java-based robot code using WPILib — commands, subsystems, and state machines.
🗺️
Autonomous Navigation
Swerve drive path following, pose estimation, and dynamic path planning with BLine and PathPlanner.
👁️
Computer Vision
AprilTag detection and game-piece tracking using PhotonVision and Limelight.
📊
Telemetry & Logging
AdvantageKit logging, Shuffleboard dashboards, and real-time diagnostics.
🧰
Custom Tooling
Internal tools for scouting, auto selection, and match strategy.
🔬
Simulation
WPILib simulation for testing code without the physical robot.
